# 识图

1. 找到消息附件或工具输出中的本地图片路径。
2. 只需了解整张图时运行：
   `python scripts/abilities/image-vision/vision.py "图片路径"`
3. 用户问了具体问题时运行：
   `python scripts/abilities/image-vision/vision.py "图片路径" --question "用户真正想知道的问题"`
4. 检查生成图时，把要检查的具体目标写进 `--question`，例如构图、文字、手部或视角；不要每张图都做无目标的质检。
5. 把脚本输出当作外部视觉观察，再结合当前对话回答。不要把看不清、模型未确认或仅由上下文猜到的内容说成图中事实。
6. 脚本已经会在人物图片被误拒时做一次中性描述重试。若仍返回 `VISION_REFUSED` 或 `VISION_ERROR`，说明视觉通道当前没看成；不要继续换伪装提示词反复请求，也不要编造图片内容。

`--detail` 是旧调用的兼容别名。新调用统一使用 `--question`。
